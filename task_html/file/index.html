HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML berfungsi sebagai kerangka dasar yang menampilkan konten seperti teks, gambar, video, dan tautan. Setiap elemen dalam HTML ditulis menggunakan tag yang mengapit konten tertentu.

CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan layout dari elemen-elemen HTML. Dengan CSS, Anda dapat mengubah warna, ukuran, font, margin, padding, dan elemen visual lainnya di halaman web. CSS memungkinkan pemisahan antara struktur dan presentasi, sehingga memudahkan pengembang web untuk mengelola tampilan tanpa harus merubah kode HTML. Beberapa contoh properti CSS yang umum adalah: color: untuk mengatur warna teks, font-size: untuk mengubah ukuran teks, background-color: untuk memberi warna latar belakang, margin dan padding: untuk mengatur jarak antar elemen. CSS dapat ditambahkan langsung di dalam file HTML atau dipisahkan ke dalam file eksternal yang terhubung menggunakan tag . Dengan memisahkan HTML dan CSS, pengembang bisa dengan mudah mengubah tampilan tanpa menyentuh struktur dasar halaman web.
